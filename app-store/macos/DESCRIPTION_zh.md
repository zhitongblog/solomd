SoloMD 是一款 macOS 上的轻量级 Markdown 编辑器。一个文件，一个窗口，专心写作。

从零开始，只为"好好写字"而生——不需要账号，不走云端，不发遥测，不做订阅。所有笔记都保存在 Mac 本地的 `.md` 与 `.txt` 纯文本文件里，任何编辑器都能打开，十年之后依然可读。

━━━━━━━━━━━━━━━━━━
核心编辑
━━━━━━━━━━━━━━━━━━

• 所见即所得——Markdown 标记符号在光标离开行后自动隐藏，写作与预览不再切换。
• 富样式渲染——标题真大、粗体真粗、代码真等宽。
• 浅色 / 深色主题，自动跟随系统外观。
• 智能识别编码——UTF-8、UTF-16、GBK、Big5 老文件也能正确打开。
• 多标签带脏标识——几份文稿并排展开。
• 文件树侧栏、大纲视图、ripgrep 风格全局搜索。

━━━━━━━━━━━━━━━━━━
Markdown 增强
━━━━━━━━━━━━━━━━━━

• KaTeX 数学公式——行内 $E=mc^2$ 与块级 $$\int$$ 实时渲染。
• Mermaid 图表——流程图、序列图、甘特图。
• 可点击的任务列表 checkbox。
• 脚注、YAML 元信息、==高亮==。
• 围栏代码块语法高亮（13+ 种语言）。
• 双栏同步滚动——预览跟随光标自动滚动。
• 点击图片或 Mermaid 图表即可放大查看。

━━━━━━━━━━━━━━━━━━
写作者工具
━━━━━━━━━━━━━━━━━━

• 专注模式——非当前段落自动变暗。
• 打字机模式——光标始终垂直居中。
• 会话恢复——自动保存，不再丢稿。
• 中文利器——简繁转换、拼音、CJK 字数统计。
• 拼写检查。
• 自定义 CSS——给预览换上你自己的样式。

━━━━━━━━━━━━━━━━━━
导出
━━━━━━━━━━━━━━━━━━

• HTML（Newsletter 级排版）
• PDF（自动渲染 KaTeX 与 Mermaid）
• Word（.docx，保留表格）
• 复制为 HTML / 纯文本 / Markdown

━━━━━━━━━━━━━━━━━━
macOS 原生体验
━━━━━━━━━━━━━━━━━━

• 通用二进制——Apple Silicon 与 Intel 都是原生。
• 文件关联——双击 .md / .txt 即可打开。
• 完整快捷键工作流——命令面板 ⌘⇧K、全局搜索 ⌘⇧F。
• 多窗口——⌘⇧N 打开新窗口编辑不同文档。
• 轻量——约 15 MB 安装包，冷启动迅速。

━━━━━━━━━━━━━━━━━━
隐私优先
━━━━━━━━━━━━━━━━━━

SoloMD 不收集任何个人数据。没有账号、没有统计、没有广告、没有跟踪。你的笔记完全保存在你自己的文件里。

SoloMD 是 MIT 协议下的开源项目。源码：github.com/zhitongblog/solomd
