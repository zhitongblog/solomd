SoloMD ist ein leichter, ablenkungsfreier Markdown-Editor für den Mac. Eine Datei. Ein Fenster. Einfach schreiben.

Von Grund auf als fokussiertes Schreibwerkzeug entwickelt — kein Konto, keine Cloud, keine Telemetrie, kein Abo. Deine Notizen bleiben als einfache .md- und .txt-Dateien auf deinem Mac, lesbar mit jedem Editor auf jeder Plattform, heute und in zehn Jahren.

KERN-EDITOR

- Live-Vorschau — Markdown-Zeichen blenden sich aus, sobald der Cursor die Zeile verlässt.
- Echtes Syntax-Styling — große Überschriften, echtes Fett, echte Monospace-Schrift für Code.
- Helles & dunkles Theme, passend zum Systemerscheinungsbild deines Mac.
- Multi-Encoding — erkennt UTF-8, UTF-16, GBK und Big5 automatisch.
- Tabs mit Änderungsindikator.
- Dateibaum-Seitenleiste, Gliederung und globale Suche.

MARKDOWN-POWER

- KaTeX-Formeln — inline und als Block, sofort gerendert.
- Mermaid-Diagramme — Flussdiagramme, Sequenzdiagramme, Gantt-Charts.
- Aufgabenlisten mit klickbaren Checkboxen.
- Fußnoten, YAML-Front-matter, Texthervorhebung.
- Syntaxhervorhebung für 13+ Sprachen.
- Synchrones Scrollen in der geteilten Ansicht.
- Bilder und Mermaid-Diagramme per Klick zoomen.

WERKZEUGE FÜR SCHREIBENDE

- Fokusmodus — inaktive Absätze werden abgedunkelt, für konzentriertes Arbeiten.
- Schreibmaschinenmodus.
- Sitzungswiederherstellung — automatisches Speichern, keine Zeile geht verloren.
- Chinesisch-Werkzeuge — Konvertierung Kurz-/Langzeichen, Pinyin, CJK-Wortzählung.
- Rechtschreibprüfung.
- Eigenes CSS.

EXPORT

- HTML (Newsletter-Qualität)
- PDF mit eingebettetem KaTeX & Mermaid
- Word (.docx) mit erhaltenen Tabellen
- Kopieren als HTML, reiner Text oder Markdown

NATIV FÜR macOS

- Universal Binary — läuft nativ auf Apple Silicon und Intel.
- Dateizuordnungen — jede .md- oder .txt-Datei per Doppelklick öffnen.
- Tastatur-zentrierter Workflow mit Befehlspalette und globaler Suche.
- Unterstützung für mehrere Fenster.
- Leichtgewichtig — schneller Kaltstart.

PRIVATSPHÄRE ZUERST

SoloMD sammelt keine persönlichen Daten. Kein Konto. Keine Analyse. Keine Werbung. Kein Tracking. Alle Notizen liegen lokal in Dateien, die dir gehören.
