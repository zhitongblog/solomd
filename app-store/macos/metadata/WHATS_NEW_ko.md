버전 4.6.2 — 지식 그래프 업데이트.

새로운 구조 레이어
• 속성 인스펙터(Cmd+Shift+I) — 유형·날짜·상태·관계 선택기로 YAML front-matter를 편집. 주석과 따옴표까지 바이트 단위로 보존하는 정밀 저장
• 유형 기반 사이드바 — 노트에 "type: Project"만 지정하면 해당 멤버들이 커스텀 아이콘·색상·열을 갖춘 접을 수 있는 전용 섹션이 됩니다
• 유형화된 관계 — belongs_to / related_to / has, 역방향 링크 자동 생성
• 관계 그래프 — 모든 노트의 나가는·들어오는 링크로 이루어진 이웃 관계를 탐색. 클릭으로 이동, Cmd+클릭으로 중심 전환
• 저장된 필터 뷰 — 재귀적인 모두/하나라도 일치 필터 빌더(유형, 상태, 날짜, 정규식)를 갖춘 사이드바 고정 뷰
• Inbox 워크플로(Cmd+E) — 노트를 정리 완료로 표시하면 자동으로 다음 노트로 이동, Inbox zero까지 한 번에
• 화이트보드 — tldraw 코드 펜스 안에 담기는 Markdown 기반 tldraw 보드. 지연 로딩으로 앱은 여전히 가볍습니다
