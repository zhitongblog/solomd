バージョン 4.6.2：ナレッジグラフ・アップデート。

新しい構造レイヤー
• プロパティインスペクタ（Cmd+Shift+I）：YAML front-matter を type / 日付 / ステータス / リレーションのピッカーで編集。必要な箇所だけを書き換えるので、コメントや引用符もバイト単位でそのまま残ります
• タイプ駆動サイドバー：ノートに「type: Project」を付けるだけで、そのメンバーがカスタムアイコン・カラー・カラム付きの折りたたみセクションとしてサイドバーに並びます
• 型付きリレーション：belongs_to / related_to / has に対応し、逆方向のリンクを自動生成
• リレーショングラフ：各ノートの発リンク・被リンクの近傍をたどって探索。クリックでジャンプ、Cmd+クリックで中心を切り替え
• 保存済みフィルタービュー：type・ステータス・日付・正規表現を all / any で自由に組み合わせられる再帰フィルタービルダー付きの常設サイドバービュー
• Inbox ワークフロー（Cmd+E）：ノートを整理済みにすると自動で次のノートへ。Inbox ゼロまで一直線
• ホワイトボード：tldraw コードフェンス内に Markdown として保存される tldraw ボード。遅延読み込みなのでアプリは軽いまま

改善と修正
• デザインシステムをゼロから刷新し、macOS のタイトルバーをツールバーと統合（信号機ボタンをツールバー内にインライン表示）
• アウトラインの見出しマーカーを設定可能に（英字 / 数字 / なし）
• Cmd+マウスホイールでアプリ全体をズーム
• 修正：日本語など CJK 入力メソッドでの入力中に、変換が途中で中断される問題を解消
• そのほか細かな修正とパフォーマンス改善を多数
