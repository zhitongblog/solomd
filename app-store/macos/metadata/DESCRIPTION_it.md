SoloMD è un editor Markdown leggero e senza distrazioni per macOS. Un file. Una finestra. Solo scrittura.

Nato fin dall'inizio come strumento di scrittura focalizzato — niente account, niente cloud, niente telemetria, niente abbonamenti. Le tue note restano in semplici file .md e .txt sul tuo Mac, leggibili da qualsiasi editor su qualsiasi piattaforma, oggi e tra dieci anni.

EDITOR ESSENZIALE

- Anteprima live — i simboli Markdown svaniscono quando il cursore lascia la riga.
- Stile di sintassi completo — titoli davvero grandi, vero grassetto, vero monospace per il codice.
- Temi chiaro e scuro che seguono l'aspetto di sistema del Mac.
- Multi-codifica — rileva automaticamente UTF-8, UTF-16, GBK e Big5.
- Schede con indicatore di modifiche.
- Albero file nella barra laterale, sommario e ricerca globale.

LA POTENZA DEL MARKDOWN

- Formule KaTeX — matematica in linea e in blocchi renderizzata all'istante.
- Diagrammi Mermaid — diagrammi di flusso, diagrammi di sequenza, Gantt.
- Elenchi di attività con caselle di controllo cliccabili.
- Note a piè di pagina, front-matter YAML, evidenziazione del testo.
- Evidenziazione della sintassi per oltre 13 linguaggi.
- Scorrimento sincronizzato in vista divisa.
- Fai clic su un'immagine o un diagramma Mermaid per ingrandirlo.

STRUMENTI PER CHI SCRIVE

- Modalità focus — i paragrafi non attivi si attenuano per favorire la concentrazione.
- Modalità macchina da scrivere.
- Ripristino della sessione — salvataggio automatico: non perdi mai una riga.
- Strumenti avanzati per il cinese — conversione semplificato/tradizionale, pinyin, conteggio parole CJK.
- Controllo ortografico.
- CSS personalizzato.

ESPORTAZIONE

- HTML (qualità da newsletter)
- PDF con KaTeX e Mermaid già integrati
- Word (.docx) con tabelle preservate
- Copia come HTML, testo semplice o Markdown

NATIVO PER macOS

- Binario universale — gira in modo nativo su Apple Silicon e Intel.
- Associazioni file — fai doppio clic su qualsiasi file .md o .txt per aprirlo.
- Flusso di lavoro centrato sulla tastiera, con palette comandi e ricerca globale.
- Supporto multi-finestra.
- Leggero — avvio a freddo rapidissimo.

PRIVACY PRIMA DI TUTTO

SoloMD non raccoglie alcun dato personale. Nessun account. Nessuna analisi. Nessuna pubblicità. Nessun tracciamento. Tutte le tue note vivono in locale, in file di tua proprietà.
