SoloMD は macOS のための、軽量で集中できる Markdown エディタです。ファイルひとつ、ウィンドウひとつ、ただ書くだけ。

ゼロから「集中して書く」ための道具として設計されました。アカウント不要、クラウド不要、テレメトリなし、サブスクリプションなし。あなたのメモは Mac 上のプレーンな .md / .txt ファイルに残り、どのプラットフォームのどのエディタからも、今日も 10 年後も読めます。

コア編集機能

- ライブプレビュー：カーソルが行を離れると Markdown 記号がフェードアウト。
- リッチな文法スタイリング：見出しは大きく、太字は太く、コードは等幅で。
- ライト / ダークテーマ、Mac のシステム外観に追従。
- 多文字エンコーディング：UTF-8、UTF-16、GBK、Big5 を自動判別。
- ダーティ表示付きタブ。
- ファイルツリーサイドバー、アウトラインビュー、グローバル検索。

Markdown パワー

- KaTeX 数式：インラインもブロックも即時レンダリング。
- Mermaid 図：フローチャート、シーケンス図、ガントチャート。
- クリック可能なチェックボックス付きタスクリスト。
- 脚注、YAML フロントマター、テキストハイライト。
- 13 以上の言語のシンタックスハイライト。
- 同期スクロールの分割ビュー。
- 画像や Mermaid 図をクリックするとズームできます。

ライターのための機能

- フォーカスモード：編集中以外の段落を薄暗くして、深い集中を。
- タイプライターモード。
- セッション復元：自動保存で一行たりとも失いません。
- 中国語パワーツール：簡体／繁体変換、ピンイン、CJK 字数カウント。
- スペルチェック。
- カスタム CSS。

エクスポート

- HTML（ニュースレター品質）
- KaTeX と Mermaid 込みの PDF
- テーブル保持の Word（.docx）
- HTML / プレーンテキスト / Markdown としてコピー

macOS ネイティブ

- ユニバーサルバイナリ：Apple Silicon でも Intel でもネイティブに動作。
- ファイル関連付け：.md / .txt ファイルをダブルクリックでそのまま開けます。
- コマンドパレットとグローバル検索による、キーボード中心のワークフロー。
- マルチウィンドウ対応。
- 軽量で、コールドスタートも高速。

プライバシーファースト

SoloMD は個人情報を一切収集しません。アカウントなし、解析なし、広告なし、トラッキングなし。あなたのメモは、あなたが所有するファイルとしてローカルに保存されます。
