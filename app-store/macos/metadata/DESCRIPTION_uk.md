SoloMD — легкий Markdown-редактор для macOS, що не відволікає. Один файл. Одне вікно. Просто пишіть.

Створений з нуля як сфокусований інструмент для письма — без акаунта, без хмари, без телеметрії, без підписки. Ваші нотатки зберігаються у звичайних файлах .md і .txt на вашому Mac, і їх відкриє будь-який редактор на будь-якій платформі — сьогодні й через десять років.

ОСНОВНЕ РЕДАГУВАННЯ

- Живий перегляд — символи Markdown зникають, щойно курсор залишає рядок.
- Насичене оформлення синтаксису — справжні великі заголовки, справжній жирний, справжній моноширинний шрифт для коду.
- Світла й темна теми, що йдуть за системним виглядом вашого Mac.
- Багато кодувань — автовизначення UTF-8, UTF-16, GBK, Big5.
- Вкладки з індикаторами незбережених змін.
- Бічна панель з деревом файлів, зміст документа та глобальний пошук.

СИЛА MARKDOWN

- Математика KaTeX — рядкові та блокові формули рендеряться миттєво.
- Діаграми Mermaid — блок-схеми, діаграми послідовностей, діаграми Ганта.
- Списки завдань із клікабельними чекбоксами.
- Виноски, YAML front-matter, виділення тексту.
- Підсвічування синтаксису для 13+ мов.
- Синхронна прокрутка розділеного перегляду.
- Клацніть зображення чи діаграму Mermaid, щоб збільшити.

ІНСТРУМЕНТИ ПИСЬМЕННИКА

- Режим фокусу — неактивні абзаци затемнюються для глибокої роботи.
- Режим друкарської машинки.
- Відновлення сесії — автозбереження, тож ви не втратите жодного рядка.
- Інструменти для китайської — конвертація спрощеної/традиційної, піньїнь, підрахунок слів CJK.
- Перевірка правопису.
- Власний CSS.

ЕКСПОРТ

- HTML (якість розсилки)
- PDF з вбудованими KaTeX і Mermaid
- Word (.docx) зі збереженням таблиць
- Копіювання як HTML, простий текст або Markdown

РІДНИЙ ДЛЯ macOS

- Універсальний бінарний файл — працює нативно на Apple Silicon та Intel.
- Асоціації файлів — двічі клацніть будь-який файл .md чи .txt, щоб відкрити його.
- Робота насамперед із клавіатури — палітра команд і глобальний пошук.
- Підтримка кількох вікон.
- Легкий — швидкий холодний старт.

ПРИВАТНІСТЬ ПОНАД УСЕ

SoloMD не збирає жодних персональних даних. Без акаунта. Без аналітики. Без реклами. Без стеження. Усі ваші нотатки живуть локально у файлах, які належать вам.
