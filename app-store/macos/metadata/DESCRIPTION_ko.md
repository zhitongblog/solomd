SoloMD는 macOS를 위한 가볍고 집중감 있는 마크다운 에디터입니다. 파일 하나, 창 하나, 그저 쓰기만 하면 됩니다.

오로지 "집중해서 쓰기" 위한 도구로 처음부터 설계되었습니다 — 계정 없음, 클라우드 없음, 텔레메트리 없음, 구독 없음. 메모는 Mac의 일반 .md/.txt 파일로 남아, 지금도 10년 뒤에도 어떤 플랫폼의 어떤 에디터에서든 읽을 수 있습니다.

핵심 편집

• 라이브 프리뷰 — 커서가 줄을 벗어나면 마크다운 기호가 자연스럽게 사라집니다.
• 풍부한 구문 스타일 — 진짜 큰 제목, 진짜 굵은 글씨, 코드용 진짜 고정폭 글꼴.
• 라이트/다크 테마 — Mac 시스템 화면 모드를 자동으로 따라갑니다.
• 다중 인코딩 — UTF-8, UTF-16, GBK, Big5 자동 감지.
• 저장 안 됨 표시가 있는 탭.
• 파일 트리 사이드바, 아웃라인 뷰, 전역 검색.

마크다운의 진가

• KaTeX 수식 — 인라인·블록 수식 모두 즉시 렌더.
• Mermaid 다이어그램 — 플로우차트, 시퀀스 다이어그램, 간트 차트.
• 클릭 가능한 체크박스가 있는 할 일 목록.
• 각주, YAML front-matter, 텍스트 하이라이트.
• 13개 이상 언어의 구문 강조.
• 분할 화면 스크롤 동기화.
• 이미지나 Mermaid 다이어그램을 클릭하면 확대됩니다.

글쓴이를 위한 도구

• 포커스 모드 — 편집 중이 아닌 단락은 어둡게 처리되어 깊이 몰입할 수 있습니다.
• 타자기 모드.
• 세션 복원 — 자동 저장으로 한 줄도 잃지 않습니다.
• 중국어 파워 도구 — 간/번체 변환, 병음, CJK 글자수 세기.
• 맞춤법 검사.
• 사용자 정의 CSS.

내보내기

• HTML(뉴스레터 품질)
• KaTeX와 Mermaid가 포함된 PDF
• 표를 보존하는 Word(.docx)
• HTML / 플레인 텍스트 / 마크다운으로 복사

macOS 네이티브

• 유니버설 바이너리 — Apple Silicon과 Intel 모두에서 네이티브로 실행.
• 파일 연결 — .md, .txt 파일을 더블클릭하면 바로 열립니다.
• 커맨드 팔레트와 전역 검색을 갖춘 키보드 중심 워크플로.
• 다중 창 지원.
• 가볍습니다 — 빠른 콜드 스타트.

프라이버시 우선

SoloMD는 어떠한 개인정보도 수집하지 않습니다. 계정 없음, 분석 없음, 광고 없음, 추적 없음. 모든 메모는 사용자 소유의 파일로 로컬에 보관됩니다.
