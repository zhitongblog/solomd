SoloMD är en lättviktig, distraktionsfri Markdown-redigerare för macOS. En fil. Ett fönster. Bara skriv.

Byggd från grunden som ett fokuserat skrivverktyg — inget konto, inget moln, ingen telemetri, ingen prenumeration. Dina anteckningar sparas som vanliga .md- och .txt-filer på din Mac, läsbara i vilken redigerare som helst på alla plattformar, i dag och om tio år.

KÄRNAN I REDIGERINGEN

- Liveförhandsgranskning — Markdown-symboler tonas bort när markören lämnar raden.
- Rik syntaxstil — riktiga stora rubriker, riktig fetstil, riktig monospace för kod.
- Ljust och mörkt tema som följer Macens systemutseende.
- Flera teckenkodningar — identifierar UTF-8, UTF-16, GBK och Big5 automatiskt.
- Flikar med osparat-indikatorer.
- Filträd i sidofältet, disposition och global sökning.

MARKDOWN-KRAFT

- KaTeX-matematik — inline- och blockmatematik renderas direkt.
- Mermaid-diagram — flödesscheman, sekvensdiagram, Gantt-scheman.
- Uppgiftslistor med klickbara kryssrutor.
- Fotnoter, YAML front-matter, textmarkering.
- Syntaxmarkering för 13+ språk.
- Synkroniserad rullning i delad vy.
- Klicka på en bild eller ett Mermaid-diagram för att zooma.

VERKTYG FÖR SKRIBENTER

- Fokusläge — icke-aktiva stycken dämpas för djupt arbete.
- Skrivmaskinsläge.
- Sessionsåterställning — sparar automatiskt så att du aldrig förlorar en rad.
- Kraftverktyg för kinesiska — konvertering förenklad/traditionell, pinyin, CJK-ordräkning.
- Stavningskontroll.
- Anpassat CSS.

EXPORT

- HTML (nyhetsbrevskvalitet)
- PDF med KaTeX och Mermaid inbakade
- Word (.docx) med bevarade tabeller
- Kopiera som HTML, oformaterad text eller Markdown

MACOS-NATIVT

- Universell binär — körs nativt på Apple Silicon och Intel.
- Filassociationer — dubbelklicka på valfri .md- eller .txt-fil för att öppna den.
- Tangentbordsfokuserat arbetsflöde med kommandopalett och global sökning.
- Stöd för flera fönster.
- Lättviktig — snabb kallstart.

INTEGRITET FRÄMST

SoloMD samlar inte in några personuppgifter. Inget konto. Ingen analys. Inga annonser. Ingen spårning. Alla dina anteckningar ligger lokalt i filer som du äger.
