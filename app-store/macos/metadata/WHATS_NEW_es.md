Versión 4.6.2 — la actualización del grafo de conocimiento.

NUEVA CAPA DE ESTRUCTURA
• Inspector de propiedades (Cmd+Shift+I) — edita el front-matter YAML con selectores de tipo/fecha/estado/relación; las escrituras quirúrgicas conservan tus comentarios y comillas byte a byte
• Barra lateral por tipos — asigna a una nota "type: Project" y sus miembros se convierten en una sección plegable de primera clase con icono, color y columnas personalizados
• Relaciones tipadas — belongs_to / related_to / has con enlaces inversos automáticos
• Grafo de relaciones — explora el vecindario de enlaces salientes y entrantes de cada nota; haz clic para navegar y Cmd+clic para pivotar
• Vistas filtradas guardadas — vistas persistentes en la barra lateral con un constructor de filtros recursivo todas/cualquiera (tipo, estado, fecha, regex)
• Flujo de Inbox (Cmd+E) — marca una nota como organizada y avanza automáticamente hasta el inbox cero
• Pizarras — tableros tldraw respaldados por Markdown dentro de un bloque de código tldraw, con carga diferida para que la app siga siendo ligera

PULIDO Y CORRECCIONES
• Sistema de diseño creado desde cero + barra de título de macOS unificada (los semáforos integrados en la barra de herramientas)
• Marcadores de título del esquema configurables (letras / números / ninguno)
• Cmd + rueda del ratón amplía toda la app
• Corregido: escribir con métodos de entrada CJK ya no se interrumpe a mitad de la composición
• Muchas correcciones menores y mejoras de rendimiento
