SoloMD is a lightweight, distraction-free Markdown editor for iPad. One file. One window. Just write.

Built from the ground up as a focused writing tool — no account, no cloud, no telemetry, no subscription. Your notes stay in plain `.md` and `.txt` files on your iPad, readable by any editor on any platform, today and in ten years.

━━━━━━━━━━━━━━━━━━
CORE EDITING
━━━━━━━━━━━━━━━━━━

• Live preview — Markdown symbols fade out after the cursor leaves the line, giving you WYSIWYG without the mode switch.
• Rich syntax styling — real big headings, real bold, real monospace for code.
• Light & Dark themes that follow iPad's system appearance.
• Multi-encoding — auto-detects UTF-8, UTF-16, GBK, Big5 so legacy files open correctly.
• Tabs with dirty indicators — keep several documents open side by side.
• File tree sidebar, outline view, and global ripgrep-style search.

━━━━━━━━━━━━━━━━━━
MARKDOWN POWER
━━━━━━━━━━━━━━━━━━

• KaTeX math — inline $E=mc^2$ and block $$\int$$ render instantly.
• Mermaid diagrams — flowcharts, sequence diagrams, Gantt charts.
• Task lists with clickable checkboxes.
• Footnotes, YAML front-matter, highlight (==mark==).
• Syntax highlighting for 13+ languages in fenced code blocks.
• Synced split-pane scroll — the preview follows your cursor.
• Tap an image or Mermaid diagram to open a pinch-zoomable overlay.

━━━━━━━━━━━━━━━━━━
BUILT FOR iPAD
━━━━━━━━━━━━━━━━━━

• 44 pt touch targets across toolbar, outline, and file tree.
• Larger default font sizes tuned for iPad screen distance.
• Split-view and multitasking friendly — the UI rescales cleanly at every width.
• Works with hardware keyboards — the full desktop shortcut set is available.
• Portrait and landscape, every orientation.
