SoloMD는 iPad를 위한 가볍고 집중감 있는 마크다운 에디터입니다. 파일 하나, 창 하나, 그저 쓰기만 하면 됩니다.

오로지 "집중해서 쓰기" 위한 도구로 처음부터 설계되었습니다 — 계정 없음, 클라우드 없음, 텔레메트리 없음, 구독 없음. 메모는 iPad의 일반 `.md`/`.txt` 파일로 남아, 지금도 10년 뒤에도 어떤 플랫폼의 어떤 에디터에서든 읽을 수 있습니다.

━━━━━━━━━━━━━━━━━━
핵심 편집
━━━━━━━━━━━━━━━━━━

• 라이브 프리뷰 — 커서가 줄을 벗어나면 마크다운 기호가 자연스럽게 사라져, 모드 전환 없이 WYSIWYG.
• 풍부한 구문 스타일 — 진짜 큰 제목, 진짜 굵은 글씨, 코드용 진짜 고정폭 글꼴.
• 라이트/다크 테마, iPad 시스템 설정을 자동으로 따라갑니다.
• 다중 인코딩 — UTF-8, UTF-16, GBK, Big5 자동 감지로 레거시 파일도 정확히 열립니다.
• 더러움(저장 안 됨) 표시가 있는 탭 — 여러 문서를 나란히 열어두세요.
• 파일 트리 사이드바, 아웃라인 뷰, ripgrep 스타일의 전역 검색.

━━━━━━━━━━━━━━━━━━
마크다운의 진가
━━━━━━━━━━━━━━━━━━

• KaTeX 수식 — 인라인 $E=mc^2$, 블록 $$\int$$ 모두 즉시 렌더.
• Mermaid 다이어그램 — 플로우, 시퀀스, 간트.
• 클릭 가능한 체크박스가 있는 할 일 목록.
• 각주, YAML 프런트매터, 하이라이트(==mark==).
• 펜스드 코드 블록에서 13개 이상 언어의 구문 강조.
• 분할 화면 스크롤 동기화 — 프리뷰가 커서를 따라갑니다.
• 이미지나 Mermaid 다이어그램을 탭하면 핀치 줌 가능한 오버레이로 열립니다.

━━━━━━━━━━━━━━━━━━
iPad를 위한 디자인
━━━━━━━━━━━━━━━━━━

• 툴바 / 아웃라인 / 파일 트리 전반에 걸친 44 pt 터치 타깃.
• iPad 보기 거리를 고려한 기본 폰트 크기.
• 분할 보기와 멀티태스킹에 친화적 — 어떤 너비에서도 UI가 깔끔히 재배치.
• 하드웨어 키보드 지원 — 데스크톱 단축키 전체 세트 사용 가능.
• 세로/가로, 모든 방향 지원.

━━━━━━━━━━━━━━━━━━
글쓴이를 위한 도구
━━━━━━━━━━━━━━━━━━

• 포커스 모드 — 편집 중이 아닌 단락은 어둡게.
• 타자기 모드 — 커서가 화면 세로 중앙에 고정.
• 세션 복원 — 자동 저장으로 한 줄도 잃지 않습니다.
• 중국어 파워 도구 — 간/번체 변환, 병음, CJK 글자수 세기.
• 맞춤법 검사.
• 사용자 정의 CSS — 자신의 프리뷰 스타일시트를 그대로 적용.
