SoloMD är en lättviktig, distraktionsfri Markdown-redigerare för iPad. En fil. Ett fönster. Bara skriv.

Byggd från grunden som ett fokuserat skrivverktyg — inget konto, inget moln, ingen telemetri, ingen prenumeration. Dina anteckningar sparas som vanliga `.md`- och `.txt`-filer på din iPad, läsbara i vilken redigerare som helst på alla plattformar, i dag och om tio år.

━━━━━━━━━━━━━━━━━━
KÄRNAN I REDIGERINGEN
━━━━━━━━━━━━━━━━━━

• Liveförhandsgranskning — Markdown-symboler tonas bort när markören lämnar raden: WYSIWYG utan lägesbyte.
• Rik syntaxstil — riktiga stora rubriker, riktig fetstil, riktig monospace för kod.
• Ljust och mörkt tema som följer iPadens systemutseende.
• Flera teckenkodningar — identifierar UTF-8, UTF-16, GBK och Big5 automatiskt så att äldre filer öppnas korrekt.
• Flikar med osparat-indikatorer — håll flera dokument öppna sida vid sida.
• Filträd i sidofältet, disposition och global sökning i ripgrep-stil.

━━━━━━━━━━━━━━━━━━
MARKDOWN-KRAFT
━━━━━━━━━━━━━━━━━━

• KaTeX-matematik — inline $E=mc^2$ och block $$\int$$ renderas direkt.
• Mermaid-diagram — flödesscheman, sekvensdiagram, Gantt-scheman.
• Uppgiftslistor med klickbara kryssrutor.
• Fotnoter, YAML front-matter, markering (==mark==).
• Syntaxmarkering för 13+ språk i kodblock.
• Synkroniserad rullning i delad vy — förhandsgranskningen följer markören.
• Tryck på en bild eller ett Mermaid-diagram för att öppna ett överlägg som du kan nypzooma.

━━━━━━━━━━━━━━━━━━
BYGGD FÖR iPAD
━━━━━━━━━━━━━━━━━━

• 44 pt stora tryckytor i verktygsfält, disposition och filträd.
• Större standardtextstorlekar anpassade för iPadens betraktningsavstånd.
• Trivs i Split View och multitasking — gränssnittet skalas snyggt i alla bredder.
• Fungerar med externa tangentbord — hela skrivbordsuppsättningen av kortkommandon finns på plats.
• Stående och liggande, alla orienteringar.

━━━━━━━━━━━━━━━━━━
VERKTYG FÖR SKRIBENTER
━━━━━━━━━━━━━━━━━━

• Fokusläge — icke-aktiva stycken dämpas för djupt arbete.
• Skrivmaskinsläge — markören hålls vertikalt centrerad.
• Sessionsåterställning — sparar automatiskt så att du aldrig förlorar en rad.
• Kraftverktyg för kinesiska — konvertering förenklad/traditionell, pinyin, CJK-ordräkning.
• Stavningskontroll.
• Anpassat CSS — använd din egen stilmall för förhandsgranskningen.

━━━━━━━━━━━━━━━━━━
EXPORT
━━━━━━━━━━━━━━━━━━

• HTML (nyhetsbrevskvalitet)
• PDF med KaTeX och Mermaid inbakade
• Word (.docx) med bevarade tabeller
• Kopiera som HTML, oformaterad text eller Markdown

━━━━━━━━━━━━━━━━━━
INTEGRITET FRÄMST
━━━━━━━━━━━━━━━━━━
