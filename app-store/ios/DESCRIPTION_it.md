SoloMD è un editor Markdown leggero e senza distrazioni per iPad. Un file. Una finestra. Solo scrittura.

Nato fin dall'inizio come strumento di scrittura focalizzato — niente account, niente cloud, niente telemetria, niente abbonamenti. Le tue note restano in semplici file `.md` e `.txt` sul tuo iPad, leggibili da qualsiasi editor su qualsiasi piattaforma, oggi e tra dieci anni.

━━━━━━━━━━━━━━━━━━
EDITOR ESSENZIALE
━━━━━━━━━━━━━━━━━━

• Anteprima live — i simboli Markdown svaniscono quando il cursore lascia la riga: WYSIWYG senza cambiare modalità.
• Stile di sintassi completo — titoli davvero grandi, vero grassetto, vero monospace per il codice.
• Temi chiaro e scuro che seguono l'aspetto di sistema dell'iPad.
• Multi-codifica — rileva automaticamente UTF-8, UTF-16, GBK e Big5, così anche i file datati si aprono correttamente.
• Schede con indicatore di modifiche — tieni aperti più documenti fianco a fianco.
• Albero file nella barra laterale, sommario e ricerca globale in stile ripgrep.

━━━━━━━━━━━━━━━━━━
LA POTENZA DEL MARKDOWN
━━━━━━━━━━━━━━━━━━

• Formule KaTeX — $E=mc^2$ in linea e blocchi $$\int$$ renderizzati all'istante.
• Diagrammi Mermaid — diagrammi di flusso, diagrammi di sequenza, Gantt.
• Elenchi di attività con caselle di controllo cliccabili.
• Note a piè di pagina, front-matter YAML, evidenziazione (==mark==).
• Evidenziazione della sintassi per oltre 13 linguaggi nei blocchi di codice.
• Scorrimento sincronizzato in vista divisa — l'anteprima segue il cursore.
• Tocca un'immagine o un diagramma Mermaid per aprirlo in un overlay con zoom a pizzico.

━━━━━━━━━━━━━━━━━━
PENSATO PER iPAD
━━━━━━━━━━━━━━━━━━

• Aree di tocco da 44 pt in barra degli strumenti, sommario e albero file.
• Caratteri predefiniti più grandi, calibrati per la distanza di lettura dell'iPad.
• Perfetto con Split View e multitasking — l'interfaccia si riadatta con eleganza a ogni larghezza.
• Funziona con le tastiere fisiche — con l'intero set di scorciatoie desktop.
• Verticale e orizzontale, in ogni orientamento.

━━━━━━━━━━━━━━━━━━
STRUMENTI PER CHI SCRIVE
━━━━━━━━━━━━━━━━━━

• Modalità focus — i paragrafi non attivi si attenuano per favorire la concentrazione.
• Modalità macchina da scrivere — il cursore resta centrato verticalmente.
• Ripristino della sessione — salvataggio automatico: non perdi mai una riga.
• Strumenti avanzati per il cinese — conversione semplificato/tradizionale, pinyin, conteggio parole CJK.
• Controllo ortografico.
• CSS personalizzato — usa il tuo foglio di stile per l'anteprima.

PRIVACY PRIMA DI TUTTO
━━━━━━━━━━━━━━━━━━
