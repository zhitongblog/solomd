SoloMD は iPad のための、軽量で集中できる Markdown エディタです。ファイルひとつ、ウィンドウひとつ、ただ書くだけ。

ゼロから「集中して書く」ための道具として設計されました — アカウント不要、クラウド不要、テレメトリなし、サブスクリプションなし。あなたのメモは iPad 上のプレーンな `.md` / `.txt` ファイルに残り、どのプラットフォームのどのエディタからも、今日も 10 年後も読めます。

━━━━━━━━━━━━━━━━━━
コア編集機能
━━━━━━━━━━━━━━━━━━

• ライブプレビュー — カーソルが行を離れると Markdown 記号がフェードアウト、モード切替なしで WYSIWYG。
• リッチな文法スタイリング — 見出しは大きく、太字は太く、コードは等幅で。
• ライト / ダークテーマ、iPad のシステム設定に追従。
• 多文字エンコーディング — UTF-8、UTF-16、GBK、Big5 を自動判別、レガシーファイルもそのまま開けます。
• ダーティ表示付きタブ — 複数のドキュメントを並べて開けます。
• ファイルツリーサイドバー、アウトラインビュー、グローバル ripgrep スタイル検索。

━━━━━━━━━━━━━━━━━━
Markdown パワー
━━━━━━━━━━━━━━━━━━

• KaTeX 数式 — インライン $E=mc^2$ もブロック $$\int$$ も即時レンダリング。
• Mermaid 図 — フローチャート、シーケンス図、ガントチャート。
• クリック可能なチェックボックス付きタスクリスト。
• 脚注、YAML フロントマター、ハイライト（==mark==）。
• フェンスドコードブロックで 13 以上の言語のシンタックスハイライト。
• 同期スクロール分割ビュー — プレビューがカーソルに追従。
• 画像や Mermaid 図をタップするとピンチズーム可能なオーバーレイで開きます。

━━━━━━━━━━━━━━━━━━
iPad のために
━━━━━━━━━━━━━━━━━━

• ツールバー / アウトライン / ファイルツリーすべてに 44 pt のタッチターゲット。
• iPad の視距離に合わせたデフォルトのフォントサイズ。
• Split View / マルチタスキング対応 — どの幅でも UI が綺麗にリサイズ。
• ハードウェアキーボード対応 — デスクトップのショートカットがすべて使えます。
• 縦・横、すべての向きに対応。

━━━━━━━━━━━━━━━━━━
ライターのための機能
━━━━━━━━━━━━━━━━━━

• フォーカスモード — 編集中以外の段落を薄暗く。
• タイプライターモード — カーソルが画面中央に固定。
• セッション復元 — 自動保存で一行たりとも失いません。
• 中国語パワーツール — 簡体／繁体変換、ピンイン、CJK 字数カウント。
• スペルチェック。
• カスタム CSS — 自前のプレビュースタイルシートをそのまま投入。

エクスポート
━━━━━━━━━━━━━━━━━━

• KaTeX & Mermaid 込みの PDF
• テーブル保持の Word（.docx）
• HTML / プレーンテキスト / Markdown としてコピー

プライバシーファースト
━━━━━━━━━━━━━━━━━━

SoloMD は個人情報を一切収集しません。アカウントなし、解析なし、広告なし、トラッキングなし。あなたのメモは、あなたが所有するファイルとしてローカルに保存されます。
