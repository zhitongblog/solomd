SoloMD — легкий Markdown-редактор для iPad, що не відволікає. Один файл. Одне вікно. Просто пишіть.

Створений з нуля як сфокусований інструмент для письма — без акаунта, без хмари, без телеметрії, без підписки. Ваші нотатки зберігаються у звичайних файлах `.md` і `.txt` на вашому iPad, і їх відкриє будь-який редактор на будь-якій платформі — сьогодні й через десять років.

━━━━━━━━━━━━━━━━━━
ОСНОВНЕ РЕДАГУВАННЯ
━━━━━━━━━━━━━━━━━━

• Живий перегляд — символи Markdown зникають, щойно курсор залишає рядок: WYSIWYG без перемикання режимів.
• Насичене оформлення синтаксису — справжні великі заголовки, справжній жирний, справжній моноширинний шрифт для коду.
• Світла й темна теми, що йдуть за системним виглядом iPad.
• Багато кодувань — автовизначення UTF-8, UTF-16, GBK, Big5, тож старі файли відкриваються коректно.
• Вкладки з індикаторами незбережених змін — тримайте кілька документів відкритими поруч.
• Бічна панель з деревом файлів, зміст документа та глобальний пошук у стилі ripgrep.

━━━━━━━━━━━━━━━━━━
СИЛА MARKDOWN
━━━━━━━━━━━━━━━━━━

• Математика KaTeX — рядкові $E=mc^2$ та блокові $$\int$$ формули рендеряться миттєво.
• Діаграми Mermaid — блок-схеми, діаграми послідовностей, діаграми Ганта.
• Списки завдань із клікабельними чекбоксами.
• Виноски, YAML front-matter, виділення (==mark==).
• Підсвічування синтаксису для 13+ мов у блоках коду.
• Синхронна прокрутка розділеного перегляду — перегляд слідує за курсором.
• Торкніться зображення чи діаграми Mermaid, щоб відкрити накладку з масштабуванням жестом «щіпка».

━━━━━━━━━━━━━━━━━━
СТВОРЕНО ДЛЯ iPAD
━━━━━━━━━━━━━━━━━━

• Сенсорні цілі 44 pt у панелі інструментів, змісті та дереві файлів.
• Більші типові розміри шрифту, підібрані під відстань до екрана iPad.
• Дружній до Split View та багатозадачності — інтерфейс акуратно масштабується на будь-якій ширині.
• Працює з апаратними клавіатурами — доступний повний набір десктопних комбінацій.
• Портретна та альбомна орієнтації — у будь-якому положенні.

ІНСТРУМЕНТИ ПИСЬМЕННИКА
━━━━━━━━━━━━━━━━━━

• Режим фокусу — неактивні абзаци затемнюються для глибокої роботи.
• Режим друкарської машинки — курсор завжди по центру екрана.
• Відновлення сесії — автозбереження, тож ви не втратите жодного рядка.
• Інструменти для китайської — конвертація спрощеної/традиційної, піньїнь, підрахунок слів CJK.
• Перевірка правопису.
• Власний CSS — підключіть свою таблицю стилів для перегляду.
