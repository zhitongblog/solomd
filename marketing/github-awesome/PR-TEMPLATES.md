```markdown
## What is SoloMD?

SoloMD is a lightweight, cross-platform Markdown editor (~15 MB installed) built with Tauri 2 and Rust.

**Key features:**
- Live preview (markers hide when cursor leaves line)
- KaTeX math rendering
- Mermaid diagrams
- Multi-encoding support (UTF-8, GBK, Big5, Shift_JIS)
- Export to HTML/PDF/DOCX
- Zero telemetry, fully offline

**Links:**
- Website: https://solomd.app
- GitHub: https://github.com/zhitongblog/solomd
- License: MIT

**Platforms:** macOS, Windows, Linux
```

## Submission Checklist

Before submitting PRs:

- [ ] Read each repo's CONTRIBUTING.md
- [ ] Check if similar entries exist
- [ ] Follow the existing format exactly
- [ ] Include all required badges/labels
- [ ] Make sure links work
- [ ] Submit one PR per repo (don't batch)

## Timing

- Submit to **awesome-markdown** first (most relevant)
- Wait 1-2 days between submissions
- Don't submit all on the same day (looks spammy)
