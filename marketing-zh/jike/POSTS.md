## 主帖

```
做了一个 Markdown 编辑器，只有 15MB。

起因是 Typora 收费了，MarkText 停更了，Obsidian 太重了。

自己用 Tauri 2 + Rust 做了一个：
• 安装包 15MB（Obsidian 是 250MB）
• 所见即所得，不用分屏预览
• GBK 自动识别，打开老文件不乱码
• 一键清除 ChatGPT 复制过来的格式垃圾
• 零遥测，完全离线

免费开源，MIT 协议。

https://solomd.app

反馈 bug 或建议欢迎直接回复 👇
```

**配图**：产品截图 + og-image-zh.png
