# 实时编辑模式 — 试一遍所有支持的语法

> 按 `Ctrl+Shift+P`（或点击工具栏中"笔尖在纸上"图标）切换到 **实时编辑** 模式。Markdown 格式会随着你的输入实时渲染 — 类似 Typora 或 Obsidian 的 Live Preview — 当光标停在某一行时，那一行的源码会重新出现。

## 链接

访问 [SoloMD 官网](https://solomd.app)，自动链接：<https://github.com/lixd/solomd>。

把光标移到链接所在的那一行，原始的 `[label](url)` 标记就会回来供你编辑；移开光标后又重新渲染成蓝色下划线。

## 行内代码与代码块

启动开发模式的命令是 `pnpm tauri dev`。

```ts
// 代码块有灰色背景，并保留语法高亮。
function greet(name: string): string {
  return `Hello, ${name}!`;
}
```

```python
# Python 也支持，由代码块的语言标签决定。
def greet(name: str) -> str:
    return f"Hello, {name}!"
```

## 光标展开 — 最神奇的部分

把光标放到上面这个标题所在的行 —— `##` 符号会重新出现，方便你编辑；离开此行后符号又会消失，恢复成渲染后的样子。

`**粗体**`、`*斜体*`、`` `代码` `` 和 `[label](url)` 都遵循同样的逻辑：光标不在那一行 → 看渲染结果；光标在那一行 → 看原始 Markdown。
