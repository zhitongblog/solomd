# Shortcuts

`Ctrl` on Linux / Windows = `Cmd` on Mac. Everything else listed here is the same.

## Files

| | |
| --- | --- |
| `Ctrl+N` | New markdown file |
| `Ctrl+Alt+N` | New plain text file |
| `Ctrl+Shift+N` | New window |
| `Ctrl+O` | Open file… |
| `Ctrl+S` | Save |
| `Ctrl+Shift+S` | Save As… |
| `Ctrl+W` | Close tab |

## View

| | |
| --- | --- |
| `Ctrl+Shift+P` | Cycle Edit / Split / Preview |
| `Ctrl+B` | Toggle file tree |
| `Ctrl+Shift+O` | Toggle outline |
| `Ctrl+\` | Split editor right |
| `Ctrl+Shift+\` | Split editor down |
| `Ctrl+Alt+→ / ←` | Focus next / previous pane |

## Search & navigate

| | |
| --- | --- |
| `Ctrl+F` | Find in editor (or in preview when in Preview mode) |
| `Ctrl+Shift+F` | Search across the open folder |
| `Ctrl+K` | Command palette |
| `F1` / `Ctrl+/` | Markdown cheat sheet |
| `Ctrl+,` | Settings |

## Export

| | |
| --- | --- |
| `Ctrl+P` | Export PDF (system print dialog) |
| `Ctrl+Shift+C` | Copy as HTML |
| `Ctrl+Alt+P` | Present slideshow |

## Tips

- The command palette (`Ctrl+K`) lists **every** command, with current shortcuts. If you forget a binding, this is faster than this cheat sheet.
- Most settings live in `Ctrl+,`, including font size, telemetry opt-out, "open file in new window", and font family.
