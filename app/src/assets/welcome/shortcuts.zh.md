# 快捷键

`Ctrl`（Linux/Windows）= `Cmd`（Mac）。其余按键全平台一致。

## 文件

| | |
| --- | --- |
| `Ctrl+N` | 新建 Markdown 文件 |
| `Ctrl+Alt+N` | 新建纯文本文件 |
| `Ctrl+Shift+N` | 新建窗口 |
| `Ctrl+O` | 打开文件… |
| `Ctrl+S` | 保存 |
| `Ctrl+Shift+S` | 另存为… |
| `Ctrl+W` | 关闭标签页 |

## 视图

| | |
| --- | --- |
| `Ctrl+Shift+P` | 编辑 / 分栏 / 预览 三档循环 |
| `Ctrl+B` | 文件树显隐 |
| `Ctrl+Shift+O` | 大纲显隐 |
| `Ctrl+\` | 向右分屏 |
| `Ctrl+Shift+\` | 向下分屏 |
| `Ctrl+Alt+→ / ←` | 焦点切到下一/上一面板 |

## 搜索 & 跳转

| | |
| --- | --- |
| `Ctrl+F` | 编辑器内查找（预览模式则是预览内查找）|
| `Ctrl+Shift+F` | 跨文件夹搜索 |
| `Ctrl+K` | 命令面板 |
| `F1` / `Ctrl+/` | Markdown 速查表 |
| `Ctrl+,` | 设置 |

## 导出

| | |
| --- | --- |
| `Ctrl+Shift+Alt+P` | 导出 PDF（系统打印对话框）|
| `Ctrl+Shift+C` | 复制为 HTML |
| `Ctrl+Alt+P` | 演讲模式 |

## 快速切换

| | |
| --- | --- |
| `Ctrl+P` | 快速切换最近文件（按最近 + 最常编辑排序）|

## 提示

- 命令面板（`Ctrl+K`）列出**所有**命令，并显示当前快捷键。一时记不起绑定？比这张表更快。
- 大部分设置在 `Ctrl+,` 里 —— 字号、遥测开关、"打开文件用新窗口"、字体等等。
