# 演讲模式

SoloMD 能把任意 Markdown 文档变成 **全屏演讲幻灯片** —— 不用 Keynote、不用 PowerPoint、不需要导出。

---

## 怎样开始

按 **`Ctrl+Alt+P`**（Mac 上 `Cmd+Alt+P`）—— 当前文档就在新全屏窗口里打开了。

也可以从命令面板（`Ctrl+K`）运行 **"Present Slideshow"** 命令。

---

## 幻灯片是怎么分页的

只包含 `---` 的行就是分页符：

```markdown
# 第一页

你好。

---

# 第二页

再见。
```

文档顶部的 front matter（`--- ... ---`）不会被当成空白幻灯片。

---

## 导航键

| 按键 | 作用 |
| --- | --- |
| → ↓ 空格 PageDown | 下一页 |
| ← ↑ PageUp | 上一页 |
| Home / End | 首页 / 末页 |
| F | 切换全屏 |
| ? | 显示快捷键 |
| Esc | 退出 |
| 鼠标点击 | 下一页 |

Vim 用户：`h j k l` 也能用。

---

## 渲染哪些内容

跟普通预览一样 —— 高亮代码块、数学公式、表格、脚注、Mermaid 图都行。图片按文件相对路径加载。

```mermaid
flowchart LR
  A[编辑] --> B[按 Ctrl+Alt+P]
  B --> C[演讲]
```

---

## 小贴士

- **首页**：单一 `# 标题` + 一两行说明，画面最干净。
- **要点**：每页 5 条以内，字号会随窗口大小自动缩放。
- **代码**：会带语法高亮。代码块过长可以拆到多页。
- **Speaker mode（演讲者模式）** 暂未实现。临时方案：拿 SoloMD 编辑窗口当备注屏。

就这些。在这份文档上试试 —— 现在就按 `Ctrl+Alt+P`。
