- **[SoloMD](https://github.com/zhitongblog/solomd)** - MCP server for one or more Markdown notes folders. 13 tools including read/search across the vault, wikilink backlinks, per-note AutoGit history (autogit_log/diff/rollback), sync status, and public share URLs. Multi-vault federation via `--workspace alias=/path`. Ships as prebuilt binaries for Linux/Windows arm64+x64 and bundled with the [SoloMD desktop editor](https://solomd.app).
