- **[SoloMD](https://github.com/zhitongblog/solomd)** — Read, search, edit, and version-control any folder of Markdown notes. 13 tools (8 generic + 5 SoloMD-specific: per-note AutoGit history, sync status, share URLs). Multi-vault federation via `--workspace alias=/path`. Ships standalone binaries (linux-x64/arm64, win-x64/arm64) and bundled with the [SoloMD desktop editor](https://solomd.app).
