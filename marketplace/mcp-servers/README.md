# `modelcontextprotocol/servers` submission

## Status

⏳ **Ready to submit.** Anthropic's curated server list lives at
[`modelcontextprotocol/servers`](https://github.com/modelcontextprotocol/servers).
We're submitting under the **🤝 Third-Party Servers** section (the
"Community Servers" subsection — we're not officially endorsed by an
organization listed in the "Official Integrations" block).

## What's here

| File | Purpose |
|---|---|
| [`entry.md`](entry.md) | The one-line markdown entry that gets appended to their README under "Community Servers". |
| [`pr-body.md`](pr-body.md) | Copy-paste PR body for the submission. |

## Submitting

1. Fork [modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers).
2. Open `README.md`. Find the "**🌎 Community Servers**" section.
3. Insert the line from [`entry.md`](entry.md) in alphabetical order
   (between the existing entries — SoloMD sorts between "Snyk" and
   "Spotify" depending on what's there at submission time).
4. Open a PR with the body from [`pr-body.md`](pr-body.md).
