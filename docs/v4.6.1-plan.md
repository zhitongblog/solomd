# SoloMD 4.6.1 — “把知识图谱层做精致” (明天发版)

> 4.6 把 7 项 Tolaria 对标功能做出来了,但都是并行快速产出的 **MVP 深度**。4.6.1 不加新功能,专做三件事,**一次性做到生产深度 + 设计系统一致 + 精致**。
> 竞品现状:Tolaria 2026.6.10(今天发)只是本地化 + Copy Git URL + 菜单修复,**无新结构性功能**。我们已整体持平并大幅领先,唯一范式差异(BlockNote 块编辑器)**不追**。

## 验收标准
- 7 个新功能从 MVP → **生产深度**(边界、性能、UX 细节),无明显粗糙处
- 全部新面板统一用 `app/src/ui/` 设计系统组件(消除 F2/F4/F5 当初的散装 CSS),视觉一致 = “精致”
- `vue-tsc` + `vite build` + `cargo test` 全绿;实时 dev-bridge QA 全过;主入口 bundle 不回归
- 明天发 **v4.6.1**(签名 mac dmg + 4×APK + AAB + CI Win/Linux,原子性后 bump 官网)

### ② 加固放大差异化(护城河)

- 确保新功能与我们独有层打通:关系/类型 → Agent 面板 + MCP 工具可读(`get_relationships`/`list_types` 之类,若低成本)
- 移动端(iPad)上新面板的响应式适配检查

### ③ Copy Note Git URL(Tolaria 2026.6.10 同款,便宜)
- 命令面板 + 笔记右键“复制 Git URL”——读 workspace 的 git remote + 当前笔记相对路径,拼 `<remote>/blob/<branch>/<path>`,复制到剪贴板
